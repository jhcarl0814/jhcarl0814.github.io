This page (https://jhcarl0814.github.io/) defines the criteria for categorizing topics and serves as a central hub, linking to my specific pages on different topics.
  • ClosedAI
    Formal science (theories can be proven or disproven);
    Engineering, deterministic tools (follow the instructions to make things work, disregard the instructions to screw things up);
    Examples:
    • Abstract concepts and reasoning/inference/operation thereon;
    • Boolean logic, sequential logic, lambda calculus, etc;
    • Structured sequential programming (without gotos), structured parallel programming (only simple control flow, e.g. call/return, parallel calls, and pipelines, possibly also fan-in and fan-out);
    • Git Help Desk: The ultimate help center for Git users, introducing concepts/terminologies in a pedagogically sound order and illustrating command options and their combinations/interactions with examples.
  • ClosedAJ
    Natural science (theories cannot be proven but can be disproven);
    Engineering, zero-order systems;
    Examples:
    • Attributes of physical entities (density, transmittance, resistivity, hardness, Young's modulus, Poisson's ratio, yield strength, Martin deflection temperature, dielectric constant, relative magnetic permeability, viscoelastic parameters, molecular structure, acid resistance, alkali resistance, UV resistance, creep properties, static friction coefficient, sliding friction coefficient, color, refractive index, molecular polarity, high-frequency loss angle, air permeability, air absorption rate, water absorption rate, color, taste, odor, price, specific gravity, molecular weight);
    • Relations between physical quantities (e.g. I=UR);
    • Unstructured sequential programming (with gotos);
  • ClosedAK
    Engineering, high-order (multivariable, multidimensional, multirelationship, multi-abstraction, etc) systems;
    Examples:
    • (After adding some inductors and capacitors to the circuit) delay, overcharge, oscillation, negative resistance area (the higher the voltage, the smaller the current);
    • Unstructured parallel programming (arbitrary control flow);
  • ClosedAL
    Social science (theories can be proven but cannot be disproven);
  • ClosedBI
    Useful conventions;
    Examples:
    • Using IPA to learn pronunciation of a foreign language to orally communicate with foreigners is useful, but when (in the future) devices for brain-to–brain communication emerges it becomes redundant;
    • IPA Help Desk: IPA chart that integrates audio, Unicode, diagrams, fonts, and user-embeddable examples, all on a single page and controllable via query parameters when sharing URLs.
  • ClosedBJ
    Theology (theories cannot be proven or disproven);
    Unuseful conventions;
    Fuzzy concepts;
    Examples:
    • Claiming "god created life" without first defining "what is god", "what is create" or "what is life";
    • Claiming "to be patriot" without first defining "what is country" (geographical location, territory, political power, sovereignty, feudal fiefdom, homeland, a walled city, an ethnic community, population & war generation device (superstition complex & military-industrial complex), financial support device (white leftist ＊ewish financial complex), science and technology innovation device (silicon valley H1B slave labor complex), vested interest stakeholders (large families/companies voting/lobbying complex), or a combination of various complex things);
